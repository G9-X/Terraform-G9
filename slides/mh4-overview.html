Week 5 MH4 API Gateway Terraform

API Gateway trước Chat Lambda

Expose Chat Lambda qua API Gateway REST API với throttling và API Key authentication — biến raw Lambda invocation thành production-grade API surface.

Vấn đề & Giải pháp

Trước (hiện tại)

  • POST /chat không yêu cầu authentication
  • Không giới hạn request rate
  • Ai biết URL đều invoke Lambda được
  • Cost không predictable (Lambda + Bedrock)
  • Vulnerable to abuse / DDoS

Sau (MH4)

  • API Key bắt buộc → 403 nếu thiếu
  • Throttling: 10 req/s, burst 20
  • Quota: 1000 requests/day
  • Lambda chỉ invoke khi auth pass
  • Cost predictable, DDoS-resistant
