Hiện trạng: API Gateway đã deploy

Terraform CI đã tạo REST API + Lambda Proxy. Dưới đây là view trên Console:

AWS API Gateway APIs › xbrain-chat-api › Resources
Resources
/ (root)
/chat
POST Lambda Proxy → xbrain-geekbrain-chat API Key: OFF
OPTIONS Mock Integration (CORS)
Vấn đề: POST /chat hiện tại không yêu cầu authentication — bất kỳ ai biết URL đều gọi được Lambda, gây risk về cost và security.

Bước 4: Deploy API

5

Deploy lại API sang stage prod

Sau khi thay đổi API Key Required, phải deploy lại để cập nhật

APIs › xbrain-chat-api › Actions › Deploy API
AWS API Gateway Deploy API
Deploy API

Deploy the current API configuration to a stage.

Sau deploy: Invoke URL: https://{api-id}.execute-api.us-west-2.amazonaws.com/prod/chat

Bước 5: Test curl — Evidence cho W5

Test 1: Authenticated → 200

$ curl -s -X POST \
  "https://abc123.execute-api.us-west-2.amazonaws.com/prod/chat" \
  -H "Content-Type: application/json" \
  -H "x-api-key: aB3kLm9xPq2rSt5uVw8yZ" \
  -d '{"question":"What is GeekBrain?"}' | jq .

{
  "statusCode": 200,
  "body": {
    "answer": "GeekBrain is a fintech startup...",
    "session_id": "default-session",
    "sources": ["company-overview.md"]
  }
}

Test 2: No API Key → 403

$ curl -s -X POST \
  "https://abc123.execute-api.us-west-2.amazonaws.com/prod/chat" \
  -H "Content-Type: application/json" \
  -d '{"question":"What is GeekBrain?"}' | jq .

{
  "message": "Forbidden"
}

# HTTP Status: 403 Forbidden
# Lambda was NOT invoked → cost saved
