    Internet Gateway — Attached to VPC
  • 📦

  • 🔒

  • 💾

    Route Tables — Public→IGW, Private App→NAT (per AZ), Private Data→isolated

  • ⚖️
    ALB SG — Inbound: 80 + 443 from 0.0.0.0/0 | Outbound: all
  • 📦
    Backend SG — Inbound: port 8080 from ALB SG only | Outbound: all
  • 💾
    RDS SG — Inbound: 3306 from Backend SG + Lambda SG | Outbound: all
  • ⚡
    Lambda SG — Inbound: none | Outbound: all (Bedrock Action Group)

  • 🎯
    Target Group — IP-based, port 8080/HTTP, health check /health
  • 👂
    HTTP Listener — Port 80 → forward to target group

  • 💻
    EC2 Auto Scaling Group — t3.small, min 1 / max 2 / desired 1, ECS-optimized AMI, IMDSv2 required
  • 📋
    Task Definition — awsvpc mode, 256 CPU / 512 MiB, ASP.NET Core backend container on port 8080
  • 🔐
    Secrets Manager ×8 — DB connection, JWT secret, Cloudinary (3), Stripe (3) — injected via ECS secrets
  • 👤
    IAM Roles — Execution Role (ECR pull + Secrets read), Task Role, Instance Role (EC2 + SSM)
  • 📊
    CloudWatch Logs — /ecs/xbrain/backend/us-dev, 14-day retention
  • 🔗
    ECS Service — Registered with ALB target group, runs in private app subnets, no public IP
